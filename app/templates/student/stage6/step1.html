테스트 데이터로 성능 평가

성능 평가란?
모델이 한 번도 본 적 없는 테스트 데이터로 얼마나 잘 예측하는지 확인합니다.
훈련 정확도가 높아도 테스트 정확도가 낮으면 과적합(Overfitting)을 의심해야 합니다.
{% set algo = idea.algorithm if idea and idea.algorithm else 'linear_regression' %}
Python 코드 (Jupyter에 붙여넣기) 
{% if algo == 'linear_regression' %}# 테스트 데이터로 성능 평가 (회귀)
from sklearn.metrics import mean_squared_error
import numpy as np

r2 = model.score(X_test, y_test)
print("R² 점수 (테스트):", round(r2, 4))

y_pred = model.predict(X_test)
rmse = np.sqrt(mean_squared_error(y_test, y_pred))
print("RMSE:", round(rmse, 4))

# R²: 1에 가까울수록 좋음 (0.7 이상이면 양호)
# RMSE: 0에 가까울수록 좋음 (예측 오차의 평균){% elif algo == 'kmeans' %}# 군집 성능 평가 (실루엣 점수)
from sklearn.metrics import silhouette_score

score = silhouette_score(X_train, model.labels_)
print("실루엣 점수:", round(score, 4))
print()
print("해석 기준:")
print("  0.7 이상 → 군집이 매우 잘 구분됨")
print("  0.5~0.7  → 군집 구분 양호")
print("  0.25~0.5 → 군집 구분 약함")
print("  0.25 미만 → 군집 구분 불명확"){% else %}# 테스트 데이터로 성능 평가 (분류)
from sklearn.metrics import accuracy_score, classification_report

y_pred = model.predict(X_test)
acc = accuracy_score(y_test, y_pred)
print("테스트 정확도:", round(acc, 4))

print("\n분류 리포트:")
print(classification_report(y_test, y_pred))

# precision: 예측이 맞을 확률
# recall: 실제값을 맞춘 비율
# f1-score: precision과 recall의 조화 평균{% endif %}

📊 성능 지표 해석 기준
{% if algo == 'linear_regression' %}
R² ≥ 0.9 → 매우 좋음 — 데이터 변동의 90% 이상을 설명
R² 0.7~0.9 → 양호 — 일반적으로 사용 가능한 수준
R² 0.5~0.7 → 보통 — 개선 여지 있음
R² < 0.5 → 낮음 — 특성 선택 또는 데이터 검토 필요
{% elif algo == 'kmeans' %}
실루엣 점수 ≥ 0.7 → 군집 매우 명확
실루엣 점수 0.5~0.7 → 군집 양호
실루엣 점수 0.25~0.5 → 군집 약함 — k값 조정 고려
실루엣 점수 < 0.25 → 군집 불명확 — 특성 재검토 필요
{% else %}
정확도 ≥ 90% → 매우 좋음
정확도 75~90% → 양호 — 실용적 수준
정확도 60~75% → 보통 — 개선 여지 있음
정확도 < 60% → 낮음 — 데이터 또는 파라미터 검토 필요
{% endif %}
