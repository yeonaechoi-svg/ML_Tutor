스텝 6 — 실습 C: 편의점 위치 K-평균 군집화

문제 정의

도시 내 편의점들의 위도, 경도 위치 데이터를 분석하여 지역별 군집(K개의 그룹)으로 나누는 모델을 만듭니다. 이 문제는 정해진 정답 없이 유사한 위치끼리 그룹을 형성하므로 K-평균 군집(K-Means Clustering)을 사용합니다.

전체 코드 (읽기 전용)

import pandas as pd
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt

# 1. 데이터 로드
df = pd.read_csv('convenience_store.csv')

# 2. 군집화에 사용할 특성 선택
X = df[['위도', '경도']]

# 3. K-평균 모델 생성 및 학습 (K=3 → 3개 군집)
model = KMeans(n_clusters=3, random_state=42)
model.fit(X)

# 4. 군집 결과 저장
df['군집'] = model.labels_
print(df.groupby('군집').size())

# 5. 시각화
colors = ['red', 'blue', 'green']
for i in range(3):
    cluster = df[df['군집'] == i]
    plt.scatter(cluster['경도'], cluster['위도'],
                c=colors[i], label=f'군집 {i}')

plt.scatter(model.cluster_centers_[:, 1],
            model.cluster_centers_[:, 0],
            c='black', marker='X', s=200, label='중심점')
plt.title('편의점 위치 군집화')
plt.legend()
plt.show()

K값의 중요성

K값을 어떻게 정할까요?
K-평균에서 K(군집 수)는 미리 지정해야 합니다. 너무 작으면 다른 특성을 가진 데이터가 같은 군집에 들어가고, 너무 크면 의미 있는 패턴을 찾기 어렵습니다.

실제로는 엘보우 방법(Elbow Method)을 사용해 최적의 K를 찾습니다.
