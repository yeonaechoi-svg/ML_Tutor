스텝 1 — 인공지능이란?

인공지능(AI)이란 무엇인가?

인공지능(Artificial Intelligence, AI)은 컴퓨터가 인간의 지능적인 행동을 흉내 내도록 만든 기술입니다. 학습, 추론, 문제 해결, 언어 이해, 시각 인식 등 인간이 수행하는 지능적 작업을 컴퓨터가 수행할 수 있도록 합니다.

지능 에이전트(Intelligent Agent)란?
주어진 환경을 인식(Perception)하고, 목표를 달성하기 위해 행동(Action)하는 시스템을 말합니다. 예를 들어, 체스 프로그램은 체스판의 상태를 인식하고 최선의 수를 선택해 행동합니다.

전통적 프로그래밍 vs 기계학습

📋 전통적 프로그래밍
• 입력: 데이터 + 규칙
• 출력: 결과
• 개발자가 규칙을 직접 코딩
• 예: 계산기, 신호등 제어
→ "규칙을 내가 만든다"
🤖 기계학습
• 입력: 데이터 + 결과
• 출력: 규칙(모델)
• 컴퓨터가 데이터에서 규칙을 학습
• 예: 스팸 필터, 추천 시스템
→ "규칙을 데이터에서 찾는다"

기계학습이 필요한 이유

스팸 메일 필터를 만든다고 가정해봅시다. 전통적 프로그래밍으로 만들려면 스팸의 특징(특정 단어, 발신자, 패턴 등)을 개발자가 직접 모두 규칙으로 만들어야 합니다. 하지만 스팸은 매일 새로운 형태로 등장합니다.

규칙이 너무 복잡하거나, 규칙이 계속 바뀌거나, 데이터에서 패턴을 찾기 어려울 때 → 기계학습이 더 효과적입니다.
