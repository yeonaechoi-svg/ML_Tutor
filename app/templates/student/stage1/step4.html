스텝 4 — 지도학습: 회귀 + 선형 회귀 알고리즘

문제 정의

자동차의 엔진 크기, 실린더 수, 연비 데이터를 바탕으로 CO₂ 배출량을 예측하는 회귀 모델을 만듭니다. 이 문제는 출력값이 연속 수치이므로 선형 회귀(Linear Regression)가 적합합니다.

데이터 탐색

데이터는 아래와 같은 구조입니다.

엔진크기실린더수연비(시내)연비(고속)CO2배출량
2.049.96.7196
2.4411.27.7221
3.5613.49.1261
...............

전체 코드 (읽기 전용)

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score

# 1. 데이터 로드
df = pd.read_csv('car_co2.csv')
print(df.head())

# 2. 특성(X)과 목표값(y) 분리
X = df[['엔진크기', '실린더수', '연비']]
y = df['CO2배출량']

# 3. 훈련/테스트 데이터 분리 (7:3)
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)
print(f'훈련 데이터: {len(X_train)}개, 테스트 데이터: {len(X_test)}개')

# 4. 선형 회귀 모델 생성 및 학습
model = LinearRegression()
model.fit(X_train, y_train)

# 5. 테스트 데이터로 예측
y_pred = model.predict(X_test)

# 6. 성능 평가
print(f'R² 점수: {r2_score(y_test, y_pred):.3f}')
print(f'평균제곱오차(MSE): {mean_squared_error(y_test, y_pred):.2f}')
