스텝 5 — 실습 B: 학생 성적 등급 분류

문제 정의

학생의 중간고사, 기말고사, 출석률, 과제 점수를 바탕으로 성적 등급(A/B/C/D/F)을 분류하는 모델을 만듭니다. 출력이 카테고리이므로 분류(Classification)를 사용합니다.

분류 알고리즘 종류: 의사결정나무(Decision Tree), 로지스틱 회귀, KNN, 랜덤 포레스트 등이 있습니다. 교과서에서는 의사결정나무를 주로 사용합니다.

의사결정나무란?

의사결정나무는 질문을 연속으로 던져 가며 결론에 도달하는 알고리즘입니다. 마치 스무고개처럼 "중간고사 점수가 80점 이상인가?" → "출석률이 80% 이상인가?" 등의 조건을 통해 등급을 결정합니다.

중간고사 ≥ 80 ?
├── Yes → 기말고사 ≥ 75 ? → Yes → A등급
└── No → 출석 ≥ 80 ? → Yes → B등급

전체 코드 (읽기 전용)

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import accuracy_score, classification_report

# 1. 데이터 로드
df = pd.read_csv('student_grade.csv')

# 2. 특성(X)과 목표값(y) 분리
X = df[['중간고사', '기말고사', '출석률', '과제점수']]
y = df['등급']  # A, B, C, D, F

# 3. 훈련/테스트 데이터 분리
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)

# 4. 의사결정나무 모델 생성 및 학습
model = DecisionTreeClassifier(random_state=42)
model.fit(X_train, y_train)

# 5. 예측
y_pred = model.predict(X_test)

# 6. 정확도 평가
print(f'정확도(Accuracy): {accuracy_score(y_test, y_pred):.3f}')
print(classification_report(y_test, y_pred))

성능 지표: 정확도(Accuracy)

정확도(Accuracy) = 전체 예측 중 맞게 예측한 비율

예) 100개 중 85개를 올바르게 분류했다면 → 정확도 = 0.85 (85%)
