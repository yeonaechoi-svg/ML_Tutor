결측치(Missing Value) 처리

방법 1 — 결측치 행 제거
결측치가 있는 행 전체를 삭제합니다. 데이터가 충분할 때 사용합니다.
방법 2 — 대표값으로 대체
평균(mean), 중앙값(median), 최빈값(mode)으로 결측치를 채웁니다.
Python — 방법 1: 결측치 행 제거 
df_clean = df.dropna()
print("전처리 전:", df.shape)
print("전처리 후:", df_clean.shape)
Python — 방법 2: 평균값으로 대체 
# 수치형 컬럼의 결측치를 평균으로 대체
df['컬럼명'] = df['컬럼명'].fillna(df['컬럼명'].mean())

# 범주형 컬럼의 결측치를 최빈값으로 대체
df['컬럼명'] = df['컬럼명'].fillna(df['컬럼명'].mode()[0])

이상치(Outlier) 처리

이상치란? 다른 데이터와 크게 벗어난 값입니다. 예: 나이가 999, 온도가 -100 등. 이상치를 그냥 두면 모델 성능이 크게 저하됩니다.
Python — IQR 방법으로 이상치 제거 
# IQR(사분위수 범위) 방법
Q1 = df['컬럼명'].quantile(0.25)
Q3 = df['컬럼명'].quantile(0.75)
IQR = Q3 - Q1

# 이상치 기준: Q1 - 1.5*IQR 미만 또는 Q3 + 1.5*IQR 초과
df_clean = df[(df['컬럼명'] >= Q1 - 1.5 * IQR) &
              (df['컬럼명'] <= Q3 + 1.5 * IQR)]

print("이상치 제거 전:", df.shape)
print("이상치 제거 후:", df_clean.shape)
Python — 박스플롯으로 이상치 시각화 
import matplotlib.pyplot as plt

plt.figure(figsize=(10, 4))
df.boxplot()
plt.xticks(rotation=45)
plt.title('컬럼별 이상치 확인')
plt.tight_layout()
plt.show()
